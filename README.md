# ResuGenie

ResuGenie is a modern, full-stack resume builder that empowers users to effortlessly create, manage, and download professional resumes. Built with the MERN stack, it provides a seamless experience with features like real-time progress saving, authentication, and dynamic preview generation.

Users can securely register, build their resumes section by section, and download them instantly in PDF or DOCX format.
